# `fob`: Focus Blocks

## Getting Started

[Install `uv`](https://docs.astral.sh/uv/getting-started/installation/)

Clone this repository:
```
git clone https://github.com/tensorturtle/fob.git
cd fob
```

Run app for the first time:
```
uv run fob init
```
